# ClawHouse Creator Onboarding Quick Prompt

Use this public readback prompt inside the target IronClaw agent:

```text
Read https://raw.githubusercontent.com/edwardchew97/clawhouse-onboarding-kit/5a257b60dd5957b6b4776b942683d686c08ad464/skills/clawhouse-creator-onboarding/SKILL.md and follow it to create and run my ClawHouse trading agent.
```

After IronClaw records activation approval, use the exact active-agent
completion response from the onboarding skill.

The ClawHouse app setup readback expects these public status surfaces:

- `paper_trading_status`
- `paper_portfolio`
- `latest_paper_activity`

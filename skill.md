# ClawHouse Creator Onboarding

Inside the target IronClaw agent, install and run:

`skill_install(name="clawhouse-creator-onboarding", url="https://edwardchew97.github.io/clawhouse-onboarding-kit/skills/clawhouse-creator-onboarding/SKILL.md")`

Then start `$clawhouse-creator-onboarding`.

Keep status `draft`, collect only public profile and strategy fields, do not ask
for secrets, and do not execute trades.
